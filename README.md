# Universal Cognitive Memory Engine

A high-performance cognitive memory system built for large-scale AI agents.

## Features
- **FastAPI Scaffolding**: High-performance REST API.
- **Memgraph Integration**: Graph-based memory storage.
- **Cognitive Strata**: Multi-layered memory processing.

## Setup
1. Clone the repository.
2. Create a virtual environment: `python -m venv venv`
3. Activate the venv: `.\venv\Scripts\Activate.ps1` (Windows) or `source venv/bin/activate` (Linux/Mac)
4. Install dependencies: `pip install -r requirements.txt`
5. Configure `.env` from `.env.example`.
6. Run the API: `uvicorn api.rest_api:app --reload`
